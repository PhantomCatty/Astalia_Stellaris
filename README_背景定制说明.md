# 内容区域背景定制指南

## 📐 设计说明

内容区域现在采用**梯形斜切遮罩**设计，左侧倾斜，右侧垂直，更具现代设计感。

## 🎨 如何定制背景图片

### 方法一：使用图片背景

1. 将你的背景图片放入 `assets` 文件夹（例如：`content_bg.jpg`）
2. 打开 `index.html`，找到 `:root` 样式部分（约第14行）
3. 修改 `--content-bg-image` 变量：

```css
--content-bg-image: url('assets/content_bg.jpg');
```

### 方法二：使用渐变色背景（当前默认）

```css
--content-bg-image: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### 方法三：使用纯色背景

```css
--content-bg-image: linear-gradient(to bottom, #7e46ff, #7e46ff);
```

## 🔧 调整梯形形状

如需调整梯形的倾斜角度，在 `.trapezoid-mask` 样式中修改 `clip-path`：

```css
/* 当前设置（左侧 8% 倾斜）*/
clip-path: polygon(8% 0%, 100% 0%, 100% 100%, 0% 100%);

/* 更倾斜的版本（左侧 15% 倾斜）*/
clip-path: polygon(15% 0%, 100% 0%, 100% 100%, 0% 100%);

/* 轻微倾斜（左侧 5% 倾斜）*/
clip-path: polygon(5% 0%, 100% 0%, 100% 100%, 0% 100%);
```

## 🎭 调整背景透明度

在 `.trapezoid-mask` 样式中修改 `opacity` 值（0-1之间）：

```css
opacity: 0.15;  /* 当前：非常淡 */
opacity: 0.3;   /* 更明显 */
opacity: 0.05;  /* 几乎透明 */
```

## 💡 推荐背景图尺寸

- 分辨率：1920x1080 或更高
- 格式：JPG（小文件）或 PNG（透明需求）
- 建议使用横向构图的图片，以适应梯形裁切

## 🌟 设计亮点

- ✅ 梯形斜切遮罩，打破常规矩形框架
- ✅ 可滚动内容区域，自定义滚动条样式
- ✅ 多层次背景叠加（背景图层 + 渐变层 + 装饰边框）
- ✅ 平滑的页面切换动画
- ✅ 响应式设计，移动端自动适配
